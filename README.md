# Justin S. Bracks professional website

Static starter site for GitHub Pages.

## Deploy

1. Create or open the repository `juzzbracks.github.io`.
2. Upload `index.html`, `style.css`, and the `assets/` folder to the repository root.
3. Add a current CV PDF at `assets/Bracks_CV.pdf`, or update the CV link in `index.html`.
4. In GitHub, go to Settings → Pages and confirm deployment from the `main` branch root.

## To customize

- Replace placeholder links in the Contact section with Google Scholar, ORCID, ADS, and GitHub links.
- Update publication statuses as papers are accepted.
- Add a `research.html` page later if the homepage gets too long.
